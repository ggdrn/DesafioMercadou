Desafio Mercadou

Sobre o desafio


    ➢ Criar um repositório público no GitHub e suba seu código para o repositório e informe-nos apenas o link do mesmo.

    ➢ Criar uma pasta chamada “perguntas” e responda cada pergunta em um arquivo Javascript, exemplo: pergunta1.js. A resposta deve estar em forma de comentário e caso haja código para executar, escreva-o normalmente.

    ➢ Os desafio e bônus não são obrigatórios porém contaram bastante para a avaliação.

    ➢ O link do repositório deve ser enviado via e-mail para contato@academiadouniversitario.com.br até a data 21/10/2019, 23:59h.

    ➢ O tempo de entrega também será avaliado. Projetos entregues após a data 21/10/2019 serão desconsiderados

Considereções do desafio


    ➢ Foi utilizado a linguagem javascript junto com o framework front-end VUE.JS para realizar as operações na página.

    ➢ Para estilizar a página foi usado o framework Bootstrap

    ➢ A resposta da pergunta 2 está respondida tanto na página html, sendo ilustrada pelos recursos da linguagem de marcação quanto nos comentários do arquivo "pegunta2.js".

    ➢ Para exibir os código javascript na página da pergunta 2, foi usado o framework PrismJs
